# OTUS SQL 2025
Проектная работа — курс OTUS «SQL для разработчиков»

---

## Задание

Разработать базу данных для **платформы управления бронированием услуг**, включающую:
- управление расписанием
- бронирование и отмену
- оплаты и возвраты
- отчётность

---

## Выбранная предметная область

**Мультимодальный транспорт** — система продажи и бронирования билетов на несколько видов транспорта (автобус, поезд, самолёт).

Пассажир может найти рейс, выбрать место, забронировать и оплатить билет. Поддерживается возврат билета, стыковочные маршруты и разные классы обслуживания.

---

## Предметная область: мультимодальный транспорт

### Сущности

- **transport_types** — виды транспорта (автобус, поезд, самолёт)
- **carriers** — перевозчики (авиакомпании, железнодорожные и автобусные компании)
- **vehicles** — транспортные средства (конкретный самолёт, вагон, автобус)
- **stations** — остановки, вокзалы, аэропорты
- **routes** — маршруты (откуда и куда следует транспорт)
- **trips** — конкретный рейс (маршрут + дата + транспортное средство)
- **seat_classes** — классы мест (эконом, бизнес, купе, плацкарт)
- **seats** — конкретные места в транспортном средстве
- **clients** — пассажиры
- **bookings** — бронирования (клиент + место + рейс)
- **tickets** — оформленные билеты
- **payments** — оплаты и возвраты
